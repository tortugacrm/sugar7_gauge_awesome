'Gauge Awesome Dashlet' module.
Copyright [2015/06/02] [Olivier Nepomiachty - SugarCRM]


1) SYNOPSIS
Sugar 7.5/7.6 dashlet.
Add some spice to your Sugar! This dashlet will bring a gauge chart in the intelligence panel.
Works with any module and any fields (standard & custom).
Choose a field for the max value and another field for the needle position.
Fields can be of any type (number, string, calculated) as long as they can be converted to numbers.
